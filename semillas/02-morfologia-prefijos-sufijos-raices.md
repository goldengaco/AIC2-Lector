# Semilla 02 — Morfología: prefijos, sufijos y raíces

> **El multiplicador.** 200 elementos morfológicos desbloquean ~5,000 palabras adicionales sin estudiarlas individualmente.
> **Tiempo:** 8-12 semanas para dominar las 200, pero el ROI empieza desde la semana 1.
> **Por qué funciona:** convierte aprendizaje de "una palabra" a "patrón de palabras". Un hispanohablante reconoce las raíces latinas/griegas instantáneamente.

---

## Cómo usar esta semilla

Cada elemento se aprende:

1. Significado en español.
2. Origen (latino / griego / germánico).
3. 5-10 palabras inglesas que lo contienen.
4. Ejercicio de inferencia: dada una palabra inventada con esa raíz, deduces significado.

## Ejercicio de inferencia (clave del método)

Una vez aprendidas raíces, debes practicar inferencia con palabras NUEVAS:

```
Palabra: "diction"
Análisis: dict (decir) + -ion (sustantivo de acción)
Inferencia: "acción de decir" → forma de hablar / pronunciación.
✓ Correcto: diction = dicción.

Palabra: "transmitter"  
Análisis: trans- (a través) + mit (enviar) + -er (cosa/persona que hace)
Inferencia: "cosa que envía a través" → transmisor.
✓ Correcto.

Palabra: "infrasonic" (palabra real, raramente conocida)
Análisis: infra- (debajo) + son (sonido) + -ic (adjetivo)
Inferencia: "relativo a debajo del sonido" → infrasónico (frecuencias por debajo del audio humano).
✓ Correcto.

Palabra inventada para entrenar: "polynomicide"
Análisis: poly- (mucho) + nom (nombre) + -i- (conector) + -cide (matar)
Inferencia: "matar muchos nombres" → no existe, pero si existiera sería algo así.
```

## Plan de 12 semanas

```
Semana 1-2:  20 prefijos negación/oposición + 10 prefijos tiempo
Semana 3-4:  20 prefijos lugar + 13 prefijos cantidad/magnitud
Semana 5-6:  35 sufijos sustantivo/adjetivo
Semana 7:    15 sufijos verbo/adverbio/otros
Semana 8-9:  25 raíces latinas top
Semana 10:   25 raíces latinas restantes
Semana 11:   25 raíces griegas top
Semana 12:   25 raíces griegas restantes + ejercicios de inferencia integrales
```

Tras 12 semanas, tu vocabulario pasivo se expande en ~5,000 palabras adicionales sin haber memorizado ni una sola más. Eso es el regalo de la morfología.
