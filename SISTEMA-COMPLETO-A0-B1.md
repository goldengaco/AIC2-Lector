### FASE 2: GRAMÁTICA SISTEMÁTICA (Semanas 9-16) — 130 HORAS
**Objetivo:** A2 → B1- (dominas tiempos, modales, voz pasiva)

#### Semana 9-10: Tiempos verbales (CRÍTICO)
```
PRESENT TENSE (4 formas):
✅ Simple present: I work, you work, he works
   Uso: habitual, verdades generales, hechos
   Ejemplo: "Machine learning works by pattern recognition"
   
✅ Present continuous: I am working, you are working
   Uso: acción en progreso ahora
   Ejemplo: "The model is training right now"
   
✅ Present perfect: I have worked, you have worked
   Uso: desde el pasado hasta ahora (reciente)
   Ejemplo: "We have implemented this feature"
   
✅ Present perfect continuous: I have been working
   Uso: acción repetida/durativa desde pasado
   Ejemplo: "I have been learning English for 3 months"

PAST TENSE (4 formas):
✅ Simple past: I worked, you worked, he worked
   Uso: acción completada en pasado
   Ejemplo: "I deployed the app last week"
   
✅ Past continuous: I was working, you were working
   Uso: acción en progreso en el pasado
   Ejemplo: "I was coding when the bug appeared"
   
✅ Past perfect: I had worked, you had worked
   Uso: anterior a otro evento pasado
   Ejemplo: "I had finished the code before the meeting"
   
✅ Past perfect continuous: I had been working
   Uso: acción durativa hasta cierto momento pasado

FUTURE TENSE (3 formas):
✅ Simple future (will): I will work, you will work
   Uso: predicción, decisión futura
   Ejemplo: "AI will change everything"
   
✅ Going to future: I am going to work, you are going to work
   Uso: plan seguro, intención clara
   Ejemplo: "I am going to learn machine learning next year"
   
✅ Present simple para futuro: The meeting is tomorrow at 3pm
   Uso: horarios, planes fijos
   Ejemplo: "The deployment starts next Monday"

TOTAL ESTRUCTURAS: 12 tiempos (es mucho pero son patrones)
TIEMPO/SEMANA: 20h (intenso, el corazón del sistema)
MÉTODO: Pattern recognition + 100 ejemplos por tiempo
```

#### Semana 11-12: Modales + Voz pasiva
```
MODAL VERBS (6 principales):
✅ Can/Could: habilidad, permiso, posibilidad
   "Can you help me?" / "I could deploy this tomorrow"
   
✅ Would: hipotético, cortesía
   "Would you like...?" / "This would be faster"
   
✅ Should: recomendación, deber
   "You should use this library" / "I should have tested more"
   
✅ Must: obligación, necesidad
   "We must fix this bug" / "You must use secure passwords"
   
✅ May/Might: posibilidad, permiso
   "This might work" / "May I ask a question?"
   
✅ Need to / Have to: obligación
   "I need to deploy this" / "You have to submit the report"

VOZ PASIVA (crítica en tech writing):
Active:   "We deploy the code"
Passive:  "The code is deployed"

Active:   "The team implemented the feature"
Passive:  "The feature was implemented by the team"

Patrón B1: Entiendes pasiva, no siempre la produces
TIEMPO/SEMANA: 15h
```

#### Semana 13-16: Condicionales + Phrasal verbs
```
CONDICIONALES (3 tipos):
Type 0 (real): If you heat water, it boils
Type 1 (posible): If you study hard, you will pass
Type 2 (improbable): If I had time, I would learn Python
Type 3 (imposible): If I had studied, I would have passed

PHRASAL VERBS (comunes en tech):
✅ Set up: configure, establish
   "Set up a server" / "The system is set up"
   
✅ Break down: explain, analyzeAnalyze, fail
   "Can you break down the algorithm?"
   
✅ Work out: calculate, solve, exercise
   "Work out the solution" / "This won't work out"
   
✅ Bring up: mention, raise
   "Bring up the issue in the meeting"
   
✅ Look into: investigate
   "Look into that bug"
   
✅ Come up with: create, invent
   "Come up with a solution"
   
✅ Roll out: deploy, release
   "Roll out the new version"
   
✅ Shut down: close, stop
   "Shut down the service"

TOTAL: 20 phrasal verbs críticos para B1
TIEMPO/SEMANA: 15h
```
