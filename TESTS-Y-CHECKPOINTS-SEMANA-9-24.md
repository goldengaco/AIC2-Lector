# 🎯 TESTS Y CHECKPOINTS — SEMANA 9-24

**Objetivo:** Evaluar progreso, identificar debilidades, certificar niveles  
**Formato:** Weekly tests (cada semana) + Monthly checkpoints (cada mes) + Level exams (cada 6 semanas)  
**Puntaje:** Sistema de puntuación transparente con rúbrica

---

## 📋 SISTEMA DE PUNTUACIÓN

```
ESCALA:
90-100 = A+ (Excelente, dominio completo)
80-89  = A  (Muy bien, pequeños ajustes)
70-79  = B  (Bien, necesita práctica)
60-69  = C  (Aceptable, requiere repaso)
<60    = F  (No aprobado, repetir semana)

META POR SEMANA: 80+ (A o mejor)
META MENUAL:     80+ promedio
META NIVEL EXAM: 85+ para pasar de nivel
```

### TEST SEMANA 9: PRESENT TENSES

**Tiempo límite:** 15 minutos  
**Total puntos:** 30

```
PARTE A: IDENTIFICACIÓN (10 preguntas × 1 punto)
Identifica el tiempo verbal

1. "The team is working on the deployment."

   b) Present Continuous ✅
   c) Present Perfect

2. "She has completed the code review."
   a) Present Simple
   b) Present Continuous
   c) Present Perfect ✅

3. "We have been testing for 3 hours."
   a) Present Perfect
   b) Present Perfect Continuous ✅
   c) Simple Past

[7 preguntas más...]

PUNTUACIÓN PARTE A: _____/10


PARTE B: FILL-IN-THE-BLANK (10 preguntas × 1 punto)
Completa con el verbo en tiempo correcto

1. "The system ________________ (run) the daily backup right now."
   Respuesta: IS RUNNING

2. "I ________________ (work) here for 5 years."
   Respuesta: HAVE BEEN WORKING

[8 preguntas más...]

PUNTUACIÓN PARTE B: _____/10


PARTE C: CORRECCIÓN (5 frases × 2 puntos)
La frase es INCORRECTA. Corrige el verbo.

1. "She are studying English this month."
   Incorrecto: "are"
   Corrección: "She IS STUDYING English this month."
   Explicación: 3ª persona singular + Present Continuous = is + -ing

2. "I has already finished my work."
   Incorrecto: "has"
   Corrección: "I HAVE already finished my work."
   Explicación: 1ª persona singular + Present Perfect = have

[3 preguntas más...]

### TEST SEMANA 14: PASSIVE VOICE + CONDITIONAL

**Tiempo límite:** 20 minutos  
**Total puntos:** 40

```
PARTE A: PASSIVE VOICE (10 preguntas × 2 puntos)
Convierte a pasiva o identifica si está correcta

1. ACTIVO: "The team built the API."
   PASIVA: "The API ________________ by the team."
   Respuesta: WAS BUILT

2. PASIVA CORRECTA? "The system is being monitored continuously."
   Respuesta: YES ✅ (is + being + monitored = Present Continuous Passive)

[8 preguntas más...]


PARTE B: CONDITIONAL TYPES (10 preguntas × 2 puntos)

Tipo 0: "If you ________________ (heat) water, it ________________ (boil)."
Respuesta: HEAT, BOILS

Tipo 1: "If we ________________ (implement) this, users ________________ (be) happy."
Respuesta: IMPLEMENT, WILL BE

Tipo 2: "If I ________________ (have) more time, I ________________ (optimize)."
Respuesta: HAD, WOULD OPTIMIZE

Tipo 3: "If we ________________ (fix) earlier, it ________________ (not happen)."
Respuesta: HAD FIXED, WOULD NOT HAVE HAPPENED

[6 preguntas más...]


PARTE C: PASIVA EN DIFERENTES TIEMPOS (10 puntos)
Escribe las frases en pasiva, manteniendo el tiempo

1. PRESENT SIMPLE PASSIVE:
   "The database maintains the data." → "The data ________________________"
   Respuesta: "The data IS MAINTAINED by the database."

2. PAST SIMPLE PASSIVE:
   "Someone discovered the vulnerability." → "The vulnerability ________________________"
   Respuesta: "The vulnerability WAS DISCOVERED."

3. PRESENT PERFECT PASSIVE:
   "They completed the update." → "The update ________________________"
   Respuesta: "The update HAS BEEN COMPLETED."

---

TOTAL PUNTUACIÓN: _____/40
```

### TEST SEMANA 16: GRAMMAR COMPREHENSIVE REVIEW

**Tiempo límite:** 30 minutos  
**Total puntos:** 50

```
PARTE A: TIMELINE EXERCISE (20 puntos)
Lee el párrafo y marca los tiempos verbales

"Our team is implementing a new system. We have been working on this project for 3 months. 
Last week, we identified an issue. While we were testing, the database crashed. 
By the time I arrived, they had already fixed it. If we had not acted quickly, 
the outage would have lasted longer. The system will be ready by next Monday. 
You should upgrade your version as soon as possible."

Identifica:
- 3 Present tenses: ________________, ________________, ________________
- 3 Past tenses: ________________, ________________, ________________
- 1 Future: ________________
- 1 Modal: ________________
- 1 Pasiva: ________________

Respuesta:
- Present: is implementing, have been working, will be
- Past: identified, were testing, crashed, had already fixed
- Future: will be
- Modal: should
- Pasiva: will be ready


PARTE B: MIXED GRAMMAR (20 puntos)
Completa con el verbo en tiempo/forma correcto

1. "The team ________________ (meet) tomorrow at 10 AM."
   Respuesta: WILL MEET o IS MEETING

2. "If I ________________ (have) known about the bug, I ________________ (warn) you."
   Respuesta: HAD KNOWN, WOULD HAVE WARNED

3. "The database ________________ (access) by unauthorized users yesterday."
   Respuesta: WAS ACCESSED (Pasiva)

4. "We ________________ (implement) this feature since January."
   Respuesta: HAVE BEEN IMPLEMENTING

5. "You ________________ (can) log in with your credentials."
   Respuesta: CAN

[5 preguntas más...]

Rúbrica:
