# 🎯 EJERCICIOS PRÁCTICOS — SEMANA 1-4 (80 VERBOS)

**Objetivo:** Dominar reconocimiento y contexto de los 80 verbos  
**Duración:** 4 semanas (Lunes-Domingo)  
**Tiempo por ejercicio:** 10-15 minutos  
**Dificultad:** A1 (principiante)

---

## 📋 ESTRUCTURA DE EJERCICIOS

Cada semana tiene 4 tipos:

1. **RECONOCIMIENTO** (5 min) — Identifica el verbo
2. **FILL-IN-THE-BLANK** (5 min) — Completa la frase
3. **CONTEXTO** (5 min) — Elige el verbo correcto
4. **TEST SEMANAL** (10 min) — Evaluación

---

## 🔵 SEMANA 1: VERBOS BÁSICOS (OPEN, CLOSE, CLICK, PRESS, TYPE, ENTER, SELECT, DELETE, SAVE, READ)

### EJERCICIO 1.1: RECONOCIMIENTO

**Instrucción:** Mira las imágenes/situaciones. ¿Qué verbo es?

```
1. Necesito hacer que un archivo desaparezca del disco duro.
   ¿Qué verbo uso?
   a) Open
   b) Delete ✅ CORRECTO
   c) Copy
   
2. Mi profesor quiere que preste atención a lo que dice.
   ¿Qué verbo es?
   a) Read
   b) Listen ✅ CORRECTO
   c) Click

3. El profesor dice: "Por favor, escribe tu nombre en el papel"
   ¿Qué verbo es?
   a) Type
   b) Write
   c) Press
   Respuesta: Ambas son válidas en contexto diferente
   - Write: con lápiz/pluma
   - Type: con teclado
   
4. Necesito ver un documento Word
   ¿Qué verbo primero?
   a) Open ✅ CORRECTO
   b) Read
   c) Click
   
5. Estoy en una app y necesito ir a la siguiente pantalla
   ¿Qué verbo?
   a) Close
   b) Press ✅ (Press next button)
   c) Enter ✅ (Enter the next screen)
   Respuesta: Ambas funcionan

PUNTUACIÓN: 5/5 = ✅ EXCELENTE
```

### EJERCICIO 1.2: FILL-IN-THE-BLANK

**Instrucción:** Completa con el verbo correcto.

```
1. "________________ the file if you want to see it."
   Opciones: Open / Close / Delete
   Respuesta: OPEN
   Explicación: Open = para ver/abrir archivos

2. "Please ________________ the window before you leave."
   Opciones: Click / Close / Enter
   Respuesta: CLOSE
   Explicación: Close = cerrar ventanas

3. "You need to ________________ your password here."
   Opciones: Type / Read / Press
   Respuesta: TYPE
   Explicación: Type = escribir en teclado

4. "________________ the button to confirm."
   Opciones: Press / Hold / Delete
   Respuesta: PRESS
   Explicación: Press = presionar un botón

5. "________________ the correct answer from the list."
   Opciones: Select / Open / Copy
   Respuesta: SELECT
   Explicación: Select = elegir/seleccionar opción

6. "I need to ________________ this document carefully."
   Opciones: Read / Type / Click
   Respuesta: READ
   Explicación: Read = leer

7. "When you finish, ________________ your work."
   Opciones: Delete / Save / Close
   Respuesta: SAVE
   Explicación: Save = guardar/salvar

8. "You must ________________ this file. Don't keep it."
   Opciones: Copy / Delete / Select
   Respuesta: DELETE
   Explicación: Delete = eliminar

9. "________________ your password and click OK."
   Opciones: Read / Enter / Type
   Respuesta: ENTER o TYPE
   Explicación: Enter = ingresar datos, Type = escribir

10. "Can you ________________ this text to your document?"
    Opciones: Copy / Press / Close
    Respuesta: COPY
    Explicación: Copy = copiar

PUNTUACIÓN: 10/10 = ✅ PERFECTO
```

### EJERCICIO 1.3: CONTEXTO

```

   Respuesta: ENTER o TYPE

   Respuesta: SAVE

   Respuesta: SELECT

PARTE C: CONTEXTO (3 preguntas)
```

```
3. En un examen, debo rodear la respuesta correcta.
   ¿Qué verbo?
   a) Highlight
   b) Circle ✅ CORRECTO
   c) Underline

4. Un formulario tiene un checkbox. Debo hacerlo.
   ¿Qué verbo?
   a) Check ✅ CORRECTO
   b) Mark
   c) Select

5. Necesito seleccionar UNA opción de varias.
   ¿Qué verbo?
   a) Pick ✅ CORRECTO
   b) Choose
   c) Select

- Choose: elección entre opciones
- Pick: seleccionar manualmente
- Select: marcar/elegir en sistema
```
