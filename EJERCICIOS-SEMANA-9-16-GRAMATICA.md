## 🔵 SEMANA 9: PRESENT TENSES (1/2)

### EJERCICIO 9.1: RECONOCIMIENTO DE TIEMPO

**Instrucción:** Lee la frase. ¿Qué tiempo es? Identifica la acción.

```
1. "I am working on the deployment right now."
   a) Present Simple
   b) Present Continuous ✅ CORRECTO
   c) Present Perfect
   
   Explicación: "am working" = ahora mismo, en este momento
   Patrón: am/is/are + -ing

2. "She works in the DevOps team."
   a) Present Simple ✅ CORRECTO
   b) Present Continuous
   c) Present Perfect
   
   Explicación: Acción habitual, no en este momento
   Patrón: verb base form (sin -ing)

3. "We have completed the migration."
   a) Present Continuous
   b) Present Perfect ✅ CORRECTO
   c) Simple Past
   
   Explicación: Acción pasada pero relación con presente
   Patrón: have/has + past participle

4. "The system is running the tests."
   a) Present Simple
   b) Present Continuous ✅ CORRECTO
   c) Present Perfect Continuous
   
   Explicación: Acción en progreso ahora
   Patrón: is + -ing

5. "I have been studying English for 2 months."
   a) Present Perfect
   b) Present Perfect Continuous ✅ CORRECTO
   c) Present Simple
   
   Explicación: Acción que COMENZÓ en el pasado y CONTINÚA ahora + duración
   Patrón: have/has + been + -ing

PUNTUACIÓN: 5/5 = ✅ EXCELENTE
```

### EJERCICIO 9.2: FILL-IN-THE-BLANK

**Instrucción:** Completa con el verbo en el tiempo correcto.

```
1. "The team ________________ (work) on the new feature right now."
   Verbo: WORK
   Tiempo: Present Continuous
   Respuesta: IS WORKING
   Patrón: is + working

2. "She ________________ (understand) the requirements."
   Verbo: UNDERSTAND
   Tiempo: Present Simple (habitual)
   Respuesta: UNDERSTANDS
   Patrón: verb + -s (3ª persona)

3. "We ________________ (implement) the API for 3 weeks."
   Verbo: IMPLEMENT
   Tiempo: Present Perfect Continuous (desde hace 3 semanas, sigue)
   Respuesta: HAVE BEEN IMPLEMENTING
   Patrón: have + been + implementing

4. "He ________________ (finish) the deployment."
   Verbo: FINISH
   Tiempo: Present Perfect (acción completada, relación con ahora)
   Respuesta: HAS FINISHED
   Patrón: has + finished

5. "The database ________________ (run) smoothly today."
   Verbo: RUN
   Tiempo: Present Continuous
   Respuesta: IS RUNNING
   Patrón: is + running

6. "I ________________ (work) as a DevOps engineer."
   Verbo: WORK
   Tiempo: Present Simple (profesión habitual)
   Respuesta: WORK
   Patrón: base form (yo soy "I")

7. "They ________________ (test) the code since this morning."
   Verbo: TEST
   Tiempo: Present Perfect Continuous
   Respuesta: HAVE BEEN TESTING
   Patrón: have + been + testing

8. "The migration ________________ (complete)."
   Verbo: COMPLETE
   Tiempo: Present Perfect
   Respuesta: HAS BEEN COMPLETED
   Patrón: has + been + completed (pasiva)

PUNTUACIÓN: 8/8 = ✅ PERFECTO

## 🟡 SEMANA 11: PAST TENSES (1/2)

### EJERCICIO 11.1: RECONOCIMIENTO

```
1. "The deployment ________________ (fail) last week."
   a) fails
   b) failed ✅ CORRECTO
   c) has failed
   
   Explicación: "last week" = pasado definido = Simple Past
   Patrón: verb + -ed

2. "We ________________ (work) when the system crashed."
   a) worked
   b) were working ✅ CORRECTO
   c) had worked
   
   Explicación: Acción EN PROGRESO en el pasado cuando pasó otra cosa
   Patrón: were + working (interrumpida por otra acción)

3. "The team ________________ (finish) the project by June."
   a) finished
   b) had finished ✅ CORRECTO
   c) was finishing
   
   Explicación: Una acción completada ANTES de otra acción pasada
   Patrón: had + past participle (Past Perfect)

4. "I ________________ (study) English for 3 years before moving."
   a) studied
   b) was studying
   c) had been studying ✅ CORRECTO
   
   Explicación: Duración EN el pasado (3 años) + acción que interrumpe
   Patrón: had + been + -ing

5. "The migration ________________ (run) when I logged in."
   a) had run
   b) was running ✅ CORRECTO
   c) ran
   
   Explicación: Acción EN PROGRESO en pasado
   Patrón: was + running

PUNTUACIÓN: 5/5 = ✅ EXCELENTE
```

   "The code ________________ (test) by them."
   Respuesta: IS BEING TESTED
   Patrón: is + being + past participle

3. "We have completed the migration." (ACTIVO)
   "The migration ________________ (complete) by us."
   Respuesta: HAS BEEN COMPLETED
   Patrón: has + been + past participle

   "The request ________________ (process) by the system."

```
